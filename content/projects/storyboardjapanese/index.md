A site for learning Japanese using examples presented in a storyboard format.

Goals

* Learn Japanese by immersing myself in jp content
* Improve programming by developing the site in [VueJS](/articles/learn-vuejs)
* Build an audience by sending content via an email newsletter
* Make money by finding what people would be willing to pay for and providing it.
  * premium content
  * tools
  * Japanese products

#### How to Start

The core of the site is going to be content.
So let's start with the basics.

Practice writing some sentences from a textbook.
Ensure I can type in Japanese.
What to do if I don't know how to type a Kanji character
